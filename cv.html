Java

2014 yılından itibaren Java ile Android uygulama geliştirmeye başladım. Android kontrolleri ve özelleştirmeleri, küçük çaplı oyun, algoritmalardan karmaşık araçlara kadar çeşitli projeler yaptım ve yer aldım.

Linux

2016 yılından itibaren Kali Linux ve Ubuntu üzerinde derinlemesine çalışarak siber güvenlik ve sistem yönetimi konularında ilerledim. Özellikle sızma testi araçları ve script geliştirme alanlarında deneyim sahibiyim.

C / C++

2018 yılından itibaren Orta seviyede C/C++ programlama bilgimle çeşitli projelerde yer aldım. Sistem programlama, gömülü sistemler ve performans kritik uygulamalar geliştirme konusunda deneyimim bulunmaktadır.

C#

2019 yılından itibaren yoğun olarak ilerleme kat ettim. Diğer diller ile komplike uygulamalar geliştirdim. Masaüstü Uygulamaları: C# ile Windows Forms ve WPF teknolojilerini kullanarak çeşitli masaüstü uygulamaları geliştirdim. Veri girişi, raporlama ve otomasyon gibi işlevselliklere sahip uygulamalar tasarladım. Görselleştirme: Veri görselleştirme konusunda çeşitli araçlar (örneğin, Chart.js) kullanarak etkileşimli grafikler ve tablolar oluşturdum. Bu sayede verileri daha anlaşılır hale getirdim. Otomasyon: Repetitive görevleri otomatikleştirmek için C# ve .NET framework'ü kullandım. Bu sayede verimlilik artışı sağladım ve hata oranını düşürdüm. Freelance Projeler: Farklı müşteriler için özel projeler geliştirerek, farklı sektörlerde ve iş gereksinimlerinde C# dilinin kullanımını deneyimledim.

Arduino ve Raspberry Pi

2019 - 2021 yılları arasında Arduino ve Raspberry Pi ile birçok projede, eğitimlerde ve yarışmalarda rol aldım. Arduino ve Raspberry Pi platformlarını kullanarak IoT projeleri geliştirdim. Sensörlerden veri toplama, veri işleme ve aktarma, uzaktan kontrol gibi konularda uygulamalar geliştirdim. Robotik projelerde Arduino ve Raspberry Pi'yi kontrol ünitesi olarak kullanarak robot kol, çizgi izleyen robot gibi sistemler geliştirdim. 2019 ODTÜ

Python

2020 yılından itibaren veri analizi, makine öğrenmesi, derin öğrenme ve yapay zeka alanlarında yoğunlaşarak Python üzerinde birçok proje geliştirdim. Özellikle derin öğrenme modelleri ve doğal dil işleme konusunda uzmanlaşmaktayım. Android ve masaüstü uygulamaları, veritabanı işlemleri ve veri görselleştirme konularında da deneyimim bulunmaktadır.

Html, Css, Java Script

2023 yılından itibaren projelerimi HTML, CSS ve Javasciprt ile destekliyorum. Bu dillerinin syntax yapısına hakimim. Standartlara uygun, temiz ve okunaklı kod yazma konusunda titizim. Responsive tasarım prensiplerine uygun olarak, farklı ekran boyutlarına uyum sağlayan web sayfaları geliştirdim. Bootstrap ve benzeri frameworkleri kullanarak hızlı ve etkili tasarımlar oluşturdum. JavaScript ile etkileşimli web sayfaları geliştirdim. DOM manipülasyonu, AJAX, Event handling gibi konularda deneyim sahibiyim. Kendi web sitem dahil olmak üzere bir çok websitesi ve kişisel projelerimde HTML, CSS ve JavaScript kullanarak farklı tasarımlar ve işlevsellikler geliştirdim.

React, Angular, Node.js

2024 yılından itibaren React, Angular ve Node.js gibi popüler teknolojilere hızlı bir şekilde hakim oldum. Özellikle backend projelerinde Node.js ile RESTful API'ler geliştirerek, veri tabanları ile etkileşim kurdum. Frontend tarafında ise React ve Angular ile dinamik ve kullanıcı dostu web uygulamaları oluşturdum. Bu teknolojileri kullanarak, hem web hem de mobil projelerde tam stack çözümler üretebilme yetkinliğine sahibim.

Flutter

2024 yıllarının sonunda popülerliğinden dolayı öğrendiğim bir framework. Java'ya benzerliği nedeniyle sytax'ına ve yapısına çabuk hakim olduğum Flutter ile de Android, iOS, Windows, Mac için native performansta mobil uygulamalar geliştirdim.
